Напишите проект, содержащий функционал работы с заметками.

Программа должна уметь:
читать список заметок (При чтении списка заметок реализовать фильтрацию по дате),
создавать заметку,
сохранять её,
редактировать заметку,
удалять заметку.

Заметка должна содержать:
идентификатор, заголовок, тело заметки и дату/время создания
или последнего изменения заметки.

Сохранение заметок необходимо сделать в формате json или csv формат
(разделение полей рекомендуется делать через точку с запятой).


Критерии оценки
---------------------
Приложение должно:
запускаться без ошибок,
должно уметь сохранять данные в файл,
уметь читать данные из файла,
выводить на экран весь список записок,
выводить на экран выбранную запись,
делать выборку по дате,
добавлять записку,
редактировать ее
и удалять.
